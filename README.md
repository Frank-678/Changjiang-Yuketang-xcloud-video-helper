# Changjiang Yuketang xcloud video helper (minimal)

给雨课堂长江站 xcloud 回放页加几个常用快捷操作：快进/快退、暂停、设置倍速，以及 MP4 直链下载按钮（blob 源不支持）。

## Features
- `A`：后退 10 秒
- `D`：前进 10 秒
- `Space`：播放/暂停
- `R`：输入倍速（如 `1.75`）
- 右下角「下载」：打开 MP4 直链（若为 `blob:` 会提示无法直接下载）

## Install
- 安装 Tampermonkey（或 Violentmonkey）
- 从 Greasy Fork 安装脚本 https://greasyfork.org/zh-CN/scripts/564640-changjiang-yuketang-xcloud-video-helper-minimal
- 或从 GitHub Raw 安装

## Scope
- 仅匹配：`https://changjiang.yuketang.cn/v2/web/xcloud/video-student/*`
